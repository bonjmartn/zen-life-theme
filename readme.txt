------------------------------
ZEN LIFE THEME
------------------------------

Thanks for using the Zen Life theme. This is a minimal blog theme with fun features that you can use for your blog, personal site, or project.

The theme's main informational page is at:
https://www.zenwebthemes.com/shop-themes/zen-life/

------------------------------
FREE AND PREMIUM VERSIONS
------------------------------

This theme comes in two versions: Free and Premium.

Free Features

- Responsive design
- Theme customizer
- Custom logo
- Social media icons in header
- Social media icons in footer
- Large photo with headline and button on homepage
- 3 featured content areas on homepage
- 3 latest blog posts automatically appear on homepage
- Call-to-action bar above the footer
- Minimal, simple, and fresh design for posts, pages, navigation, and archives
- Documentation

Premium Features

The Premium version has all of the free features, plus:

- Choose your own accent color
- Choose from 12 Google Fonts for headings and paragraph text
- Choose paragraph text size
- Choose colors for text and headings
- Custom sidebar widget for social media icons
- Custom "About Me" / "About Us" sidebar widget
- Different sidebars for Posts and Pages
- End of Posts widget
- Choose from Left Sidebar, Right Sidebar, and Full Width page layouts
- Tech support
